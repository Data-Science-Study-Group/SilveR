InVivoStat Correlation Analysis

Responses and transformations

This module assesses the correlation between the response variables Resp 1, Resp 2, Resp 3 and Resp 4. The responses have been ArcSine transformed prior to analysis.

Analysis description

The test statistic is based on Pearson's product moment correlation coefficient cor(x, y) and follows a t distribution with length(x)-2 degrees of freedom if the samples follow independent normal distributions, see Snedecor and Cochran (1989).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

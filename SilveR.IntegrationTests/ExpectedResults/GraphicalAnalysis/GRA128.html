Selected variables

The variable Resp 1 has been selected as the response and the variable Time 1 has been selected to define the X-axis. The variable Resp 1 has been Square Root transformed. The variable Cat1 has been selected as the categorisation factor. The variable Anima 1 defines the case profiles.

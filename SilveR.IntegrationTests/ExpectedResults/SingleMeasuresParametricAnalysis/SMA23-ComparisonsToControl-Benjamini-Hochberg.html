Analysis description

The data were analysed using a 1-way ANOVA approach, with Treat1 as the treatment factor. This was followed by comparisons of the predicted means of the Treat1 factor back to the control group mean using Benjamini-Hochberg's procedure.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

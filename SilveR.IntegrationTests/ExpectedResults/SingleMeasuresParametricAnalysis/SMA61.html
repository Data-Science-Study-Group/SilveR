Analysis description

The data were analysed using a 2-way ANOVA approach, with Treat9 and Treat10 as the treatment factors. This was followed by all pairwise comparisons between the predicted means of the Treat9 factor using Hochberg's procedure.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

Analysis description

The data were analysed using a 2-way ANCOVA approach, with Treat1 and Treat2 as the treatment factors and Cov 1 as the covariate.This was followed by comparisons of the predicted means of the Treat1 factor back to the control group mean using Hochberg's procedure. This was followed by all pairwise comparisons between the predicted means of the Treat1 factor using Hochberg's procedure. The response was Square Root transformed prior to analysis to stabilise the variance.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

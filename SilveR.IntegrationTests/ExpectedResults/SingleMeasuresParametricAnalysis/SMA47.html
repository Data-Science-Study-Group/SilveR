Analysis description

The data were analysed using a 3-way ANOVA approach, with Treat1, Treat2 and Treat3 as the treatment factors and Blo ck1 and Blo ck2 as the blocking factors. This was followed by all pairwise comparisons between the predicted means of the Treat1 * Treat2 interaction using Hochberg's procedure. The response was Square Root transformed prior to analysis to stabilise the variance.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

Analysis description

The data were analysed using a 1-way ANOVA approach, with Treat2 as the treatment factor and Blo ck1 and Blo ck2 as the blocking factors. This was followed by comparisons of the predicted means of the Treat2 factor back to the control group mean using Hommel's procedure. This was followed by all pairwise comparisons between the predicted means of the Treat2 factor using Hommel's procedure. The response was Square Root transformed prior to analysis to stabilise the variance.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

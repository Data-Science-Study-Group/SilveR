Analysis description

The data were analysed using a 1-way ANCOVA approach, with Treat1 as the treatment factor and Cov 1 as the covariate.This was followed by comparisons of the predicted means of the Treat1 factor back to the control group mean using Dunnett's procedure. This was followed by all pairwise comparisons between the predicted means of the Treat1 factor using Tukey's procedure. The response was Log10 transformed prior to analysis to stabilise the variance.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

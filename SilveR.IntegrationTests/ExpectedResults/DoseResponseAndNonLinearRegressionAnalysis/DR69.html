Description of the analysis results

The data was analysed using non-linear regression. The estimate of the minimum of the curve is 0.153. The estimate of the maximum of the curve is 0.993. The estimate of the slope coefficient of the curve is 1.077. The estimate of the logED50 of the curve is -2.536. The back-transformed estimate of the ED50 of the curve is 0.079.

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

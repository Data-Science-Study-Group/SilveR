InVivoStat Paired t-test Analysis and Within-Subject Analysis

Response and covariance structure

The Resp 1 response is currently being analysed by the Paired t-test Analysis and Within-Subject Analysis module.

The response has been Loge transformed prior to analysis.

The repeated measures mixed model analysis is using the compound symmetric covariance structure to model the within-subject correlations. When using this structure you are assuming sphericity and also that the variability of responses is the same at each level of Day4, see Pinherio and Bates (2002). These assumptions may not hold in practice.

Analysis description

The data were analysed using a repeated measures mixed model approach, with treatment factor Day4 and Block1 as the blocking factor. This was followed by planned comparisons on the predicted means to compare the levels of the Day4. The response was Loge transformed prior to analysis to stabilise the variance.

The compound symmetric covariance structure was used to model the within-subject correlations. When using this structure we assumed that the variability of the responses was the same at each level of Day4 and the correlation between responses from any pair of levels of Day4 is the same.

A full description of mixed model theory, including information on the R nlme package used by InVivoStat, can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

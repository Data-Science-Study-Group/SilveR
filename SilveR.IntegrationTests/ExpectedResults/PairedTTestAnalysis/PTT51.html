Analysis description

The data were analysed using a repeated measures mixed model approach, with treatment factor Day4 and Block1 as the blocking factor. This was followed by planned comparisons on the predicted means to compare the levels of the Day4. The response was Square Root transformed prior to analysis to stabilise the variance.

The unstructured covariance structure allowed the variability of the responses to be different, depending on the level of Day4. This structure also allowed the correlation between responses from any pair of levels of Day4 to be different. While this approach is the most general it should be used with care when there are few subjects, as many parameters need to be estimated. These estimates may not be very reliable.

A full description of mixed model theory, including information on the R nlme package used by InVivoStat (beta version), can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

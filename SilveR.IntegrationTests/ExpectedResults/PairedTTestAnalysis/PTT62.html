Analysis description

The data were analysed using a repeated measures mixed model approach, with treatment factor PVTestDay1 and PVTestCOV1a as the covariate.This was followed by planned comparisons on the predicted means to compare the levels of the PVTestDay1.

The compound symmetric covariance structure was used to model the within-subject correlations. When using this structure we assumed that the variability of the responses was the same at each level of PVTestDay1 and the correlation between responses from any pair of levels of PVTestDay1 is the same.

A full description of mixed model theory, including information on the R nlme package used by InVivoStat (beta version), can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).

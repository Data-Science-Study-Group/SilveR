InVivoStat Multivariate Principal Components Analysis

Description

The following responses are included in the PCA analysis: Sepal length,Sepal width,Petal width,Petal length. The responses have been Square Root transformed prior to analysis. Additionally, the responses have been centered at zero and are scaled to have unit variance.

Analysis description

The data were analysed using a 4-way repeated measures mixed model approach, with Treat 1, Treat 2 and Treat3 as treatment factors, Day 1 as the repeated factor and Block1 and Block2 as blocking factors. The response was Square Root transformed prior to analysis to stabilise the variance.

The compound symmetric covariance structure was used to model the within-subject correlations. When using this structure we assumed that the variability of the responses was the same at each level of Day 1 and the correlation between responses from any pair of levels of Day 1 is the same.

A full description of mixed model theory, including information on the R nlme package used by InVivoStat (beta version), can be found in Venables and Ripley (2003) and Pinherio and Bates (2002).

For more information on the theoretical approaches that are implemented within this module, see Bate and Clark (2014).
